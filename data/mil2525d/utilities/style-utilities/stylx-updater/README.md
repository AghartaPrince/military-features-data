# stylx-updater

## Purpose

* This is a sample app/Pro plugin demonstrating how to perform batch operations (read/update) on a style file
* The current example shows how to check and update a symbol property in the style

## Requirements

* Visual Studio 2013+
* Pro 1.2+ 
* Pro SDK
